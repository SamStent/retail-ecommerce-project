## 📄 E-commerce Sales Dataset Documentation

### 📘 Dataset Overview

This dataset contains **100 rows** of clean and structured e-commerce transaction records. It simulates sales data from an online retail platform (similar to Amazon, Flipkart, or Meesho). The data includes customer demographics, product information, pricing, location, payment method, and ratings.

It is completely **free of missing or duplicate values**, making it ideal for immediate use in exploratory data analysis (EDA), visualization, business analytics, and machine learning tasks.

---

### 📊 Dataset Features

| Column Name        | Description                                       | Data Type   |
| ------------------ | ------------------------------------------------- | ----------- |
| `Order ID`         | Unique identifier for each transaction            | String      |
| `Customer ID`      | Unique ID for each customer                       | String      |
| `Gender`           | Gender of the customer (Male/Female)              | Categorical |
| `Age`              | Age of the customer                               | Integer     |
| `Product Category` | High-level category (Electronics, Clothing, etc.) | Categorical |
| `Product Name`     | Specific product name under the category          | Categorical |
| `Quantity`         | Number of items purchased                         | Integer     |
| `Price`            | Price per item (in INR)                           | Float       |
| `Order Date`       | Purchase date                                     | Date        |
| `Payment Method`   | Payment type used (UPI, COD, Netbanking, etc.)    | Categorical |
| `City`             | City from which the order was placed              | Categorical |
| `Rating`           | Product rating (1 to 5 stars)                     | Integer     |
